# AI Inference Performance Lab

A systems performance lab focused on AI inference, CUDA, C++ runtime systems, model serving, latency measurement, throughput benchmarking, and GPU accelerated computing.

This project tracks practical experiments around inference performance: CPU vs GPU execution, memory transfer overhead, batching, latency percentiles, model serving, observability, and runtime optimization.

## Focus Areas

1. AI inference performance
2. CUDA and GPU accelerated computing
3. C++ runtime systems
4. Model serving and request latency
5. Throughput and percentile based benchmarking
6. Containers, observability, and scalable infrastructure

## Why This Exists

Modern AI systems are not only about model quality. Production inference depends on latency, throughput, memory movement, batching, GPU utilization, runtime overhead, and reliable serving infrastructure.

This lab is my hands on path toward AI inference performance engineering.
